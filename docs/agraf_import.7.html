NAME

agraf_import - import CSV files into the MySQL database

SYNOPSIS

agraf_import.py [args]

DESCRIPTION

agraf_import(7) loads the CSV files into the local MySQL database. The CSV files must be already unpacked into the secure directory. (See secure_file_priv in the /etc/my.cnf file.)

During the import all tables will be re-created. The import is done using the MySQL load data infile command.

After downloading the agraf_import.tar.gz, you have to unpack it in a new directory. This file contains both Python import scripts and all DDL statements for create a new database and all corresponding tables.

OPTIONS:

agraf_import.py --config ConfigFile --database Database --user User --password Password --import_dir Directory [--verbose]

All options can be specifed both in the configuration JSON file or on the command line.

  • {-c | --config} ConfigFile: JSON configuration file.

  • {-d | --database} Database: MySQL database.

  • {-u | --user} User: MySQL database user.

  • {-p | --password}Password: MySQL user password.

  • {-i | --import_dir} Directory: Import directory with CSV files.

EXAMPLES

Usually you should create a JSON configuration file (mysql.json) and use it for the import.

{"database": "grafana", "user": "grafana", "password": "UserPassword", "import_dir": "/agraf/import" }

./agraf_import.py -c $HOME/mysql.json

You can also specify all parameters on the command line:

./agraf_import.py -d grafana -u grafana -p password -i /agraf/import
